st is a simple terminal implementation for X.

# **Preview**

# **Quick Keybind**
```
# Copy & Paste
Ctrl + Shift + C | Copy Selected
Ctrl + Shift + V | Paste

# Zoom
Ctrl + + | Zoom In
Ctrl + - | Zoom Out
Ctrl + 0 | Reset Zoom

# Scroll
Shift + Page_Up | Scroll Up
Shift + Page_Down | Scroll Down
```
