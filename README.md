# **Quick Keybind**
```
# Copy & Paste
Ctrl + Shift + C | Copy Selected
Ctrl + Shift + V | Paste

# Zoom
Ctrl + + | Zoom In
Ctrl + - | Zoom Out
Ctrl + 0 | Reset Zoom

# Scroll
Ctrl + Shift + Page_Up | Scroll Up
Ctrl + Shift + Page_Down | Scroll Down
```
